# AD 密码管理系统 - 项目结构说明

## 📁 项目概览

**项目名称:** AD 密码管理系统 (Active Directory Password Management System)  
**技术栈:** Flask + Vue.js + LDAP/AD  
**用途:** 企业 AD 域用户密码自助管理、MFA 认证、用户同步

## 🎯 数据流

### 用户登录流程

```
前端 (Login.vue)
    ↓ (提交用户名密码)
后端 (ldap_auth.py)
    ↓ (判断用户类型)
    ├─ admin 用户 → 本地数据库认证 (bcrypt)
    └─ AD 用户 → LDAP 认证 (UPN)
        ↓ (认证成功)
    设置会话 (session)
        ↓
    跳转页面 (Dashboard/User Index)
```

---

### AD 用户同步流程

```
前端 (UserManage.vue)
    ↓ (点击同步按钮)
后端 (admin.py)
    ↓ (调用 LDAP 服务)
ldap_service.py
    ↓ (连接 AD 服务器)
    ├─ 搜索用户 (使用 UPN)
    ├─ 获取用户属性
    └─ 保存到数据库
        ↓
    返回同步结果
        ↓
    前端显示用户列表
```

---

### 密码修改流程

```
前端 (ChangePassword.vue)
    ↓ (提交新密码)
后端 (user.py)
    ↓ (验证登录状态)
    ├─ AD 用户 → LDAPS 修改 AD 密码
    └─ admin 用户 → 修改本地密码 (bcrypt)
        ↓
    返回修改结果
        ↓
    前端显示成功提示
```
