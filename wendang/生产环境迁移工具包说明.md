## ⚙️ 配置说明

### .env.production.example - 生产环境配置模板

**关键配置项：**

```bash
# 安全配置
FLASK_ENV=production
FLASK_DEBUG=False
SECRET_KEY=<强随机密钥>

# 数据库配置
DATABASE_URL=sqlite:///backend/instance/ad_password.db
# 或 PostgreSQL
# DATABASE_URL=postgresql://user:pass@localhost:5432/ad_password_db

# LDAP 配置
LDAP_HOST=192.168.1.226
LDAP_PORT=389
LDAP_BASE_DN=DC=helixon,DC=com
LDAP_ADMIN_DN=CN=Administrator,CN=Users,DC=helixon,DC=com
LDAP_PASSWORD=<LDAP 管理员密码>

# MFA 配置
MFA_REQUIRED_FOR_PASSWORD_CHANGE=True
MFA_REQUIRED_FOR_LOGIN=False

# 日志配置
LOG_LEVEL=INFO  # 生产环境建议 INFO 或 WARNING
LOG_FILE=logs/app.log
```

**使用步骤：**
1. 复制模板：
   ```bash
   cd backend
   copy .env.production.example .env.production
   ```

2. 编辑 `.env.production` 文件

3. 根据实际情况修改配置项

4. 确保 `.env.production` 不被提交到版本控制（已添加到 .gitignore）

## 🔧 故障排查

### 问题 1：清理后无法登录

**症状：** 迁移后无法使用 admin 账号登录

**解决方案：**
```python
# 重新创建 admin 账号
cd backend
python -c "
from models.models import db, User
from app import create_app
import bcrypt

app = create_app()
app.app_context().push()

# 检查是否已有 admin
admin = User.query.filter_by(username='admin').first()
if not admin:
    # 创建 admin 账号
    admin = User(
        username='admin',
        password_hash=bcrypt.hashpw(b'admin123', bcrypt.gensalt()).decode('utf-8'),
        role='admin'
    )
    db.session.add(admin)
    db.session.commit()
    print('✓ admin 账号创建成功')
else:
    print('✓ admin 账号已存在')
"
```

### 问题 2：MFA 二维码不显示

**症状：** MFA 设置页面显示"服务器内部错误"

**解决方案：**
```bash
# 安装 Pillow 依赖
cd backend
.venv\Scripts\pip install Pillow

# 重启后端服务
```

### 问题 3：LDAP 连接失败

**症状：** 域配置测试连接失败

**解决方案：**
1. 检查 LDAP 配置
2. 确认 LDAP 密码是明文（不是 bcrypt 加密）
3. 测试 LDAP 连接：
   ```bash
   python scripts\test_ldap_connection.py
   ```

### 问题 4：健康检查分数低

**症状：** health_check.py 评分低于 60 分

**解决方案：**
1. 查看检查报告中的问题列表
2. 根据建议逐项优化
3. 重新执行健康检查
